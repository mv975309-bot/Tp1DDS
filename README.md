TP1 – Portfolio Web

Trabajo práctico de la materia Desarrollo de Software – UTN.

Este proyecto consiste en la creación de un portfolio personal web estático, desarrollado con HTML, CSS y JavaScript.

Descripción

El objetivo del trabajo es diseñar y desarrollar una página web personal que funcione como portfolio profesional, aplicando conceptos básicos de desarrollo web:

Estructura semántica con HTML
Estilos y diseño responsive con CSS
Interactividad básica con JavaScript

Tecnologías utilizadas
HTML5
CSS3

Estructura del proyecto
Tp1DDS/
│
├── html/        → Páginas HTML del sitio
├── css/         → Hojas de estilo
├── js/          → Scripts JavaScript
├── Imagenes/    → Recursos gráficos
└── tp1.pdf      → Enunciado del trabajo práctico

Cómo ejecutar el proyecto
Clonar el repositorio:
git clone https://github.com/mv975309-bot/Tp1DDS.git
Abrir el proyecto en el explorador de archivos.
Ejecutar el archivo principal desde la carpeta html (doble click).

No requiere instalación ni servidor.

Objetivos académicos
Comprender la estructura de un sitio web estático
Aplicar buenas prácticas de organización de archivos
Introducción al diseño responsive
Publicación de proyectos en GitHub
